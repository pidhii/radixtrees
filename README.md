# radixtrees

This repository is a sandbox for implementation of [radix tries](https://en.wikipedia.org/wiki/Radix_tree).

Only those variants of tries that I use for other projects can be found here.
Similarly, whatever can be found here will have its interface developed no further than my needs require it to be.
